A high-fidelity, professional research portfolio designed for academic selection committees and industrial R&D partners. This project showcases a "Dark Academic" aesthetic with a specialized focus on Chemical Engineering and Process Control.

## 🌟 Key Features

### 🖼️ Scholarly Visual Abstracts
Integrated authentic graphical abstracts for peer-reviewed publications (PEEK Engineering, Uranium Remediation). Optimized for technical legibility with a dedicated 320px cadre and `object-fit: contain` rendering.

### 🌙 Dark-First Professional Aesthetic
A premium Red, White, and Black color palette set to **Dark Mode by default**. Designed to minimize eye strain during long review sessions while highlighting technical schematics and data.

### ⚡ Automated Engineering Profile
- **Dynamic Age Calculation**: Automatically updates according to birthdate using robust date-diff logic.
- **Iconic Technical Grid**: Specialized icons for PyTorch (AI), LaTeX, Hysys, and analytical instruments (HPLC, UV-Vis).
- **Responsive Timeline**: Interactive resume for Education (PhD candidate,ranked 1st) and Industrial Experience.

## 🛠️ Technical Stack

- **Core**: React 18, React Router 6.
- **Styling**: Styled-Components (Custom Theme Engine).
- **Animations**: Framer Motion & AOS (Animate On Scroll).
- **Icons**: React Icons (SimpleIcons, BootstrapIcons, FontAwesome).

## 🚀 Local Development

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/aimen-researcher/research-portfolio.git
   cd research-portfolio
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start Development Server**:
   ```bash
   npm start
   ```

## 📦 Deployment

This project is configured for one-click deployment to **GitHub Pages**:

```bash
npm run deploy
```
